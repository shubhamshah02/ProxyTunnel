# ProxyTunnel — Privacy & Legal Site

This repository hosts the privacy policy and terms of use for **ProxyTunnel**, an iOS proxy manager for developers and QA engineers.

## Pages

| Page | URL |
|---|---|
| Home | `index.html` → `proxytunnel.app` |
| Privacy Policy | `privacy.html` → `proxytunnel.app/privacy` |
| Terms of Use | `terms.html` → `proxytunnel.app/terms` |

## GitHub Pages

Hosting via GitHub Pages. To set up:

1. Go to **Settings → Pages** in this repository.
2. Set **Source** to `main` branch, `/ (root)`.
3. Save. GitHub will publish the site at your configured domain.

## Custom Domain

The app's `AboutView.swift` links to `https://proxytunnel.app/privacy` and `https://proxytunnel.app/terms`. Configure `proxytunnel.app` as a custom domain in GitHub Pages settings.
